Adds admin features from bbpress admin in the wordpress admin section

== Description ==

The plugin will add bbpress admin sections to the wordpress admin section. That way the user will be able to manage both installation from a single admin section.

<strong>The plugin has been tested with bbpress 0.9.0.1</strong>

The sections from bbpress are moved into wordpress as follows:

* <strong>bbPress Admin > Design > Themes</strong> moved to <strong>Wordpress Admin > Design > BBPress Theme</strong>
* <strong>bbPress Admin > Manage > Topics</strong> moved to <strong>Wordpress Admin > Manage > Forum Topics</strong>
* <strong>bbPress Admin > Manage > Posts</strong> moved to <strong>Wordpress Admin > Manage > Forum Posts</strong>
* <strong>bbPress Admin > Manage > Forums</strong> moved to <strong>Wordpress Admin > Manage > Forums</strong>
* <strong>bbPress Admin > Settings > General</strong> moved to <strong>Wordpress Admin > Settings > BBPress General Setting</strong>


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload all the files in a subdirectory (e.g. bbpress-admin) in the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
